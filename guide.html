Ручная установка

1. Скопируйте папку расширения в постоянное место, например:

%LOCALAPPDATA%\PENA Agency\Extension

2. Откройте свойства ярлыка Bitrix24 → поле «Объект» → добавьте в конец строки:

--disable-extensions-except="%LOCALAPPDATA%\PENA Agency\Extension" --load-extension="%LOCALAPPDATA%\PENA Agency\Extension"
  1. 1
    В Finder откройте папку installers/macos/ внутри папки расширения.
  2. 2
    Откройте Терминал: нажмите ⌘ Пробел, наберите Terminal и нажмите Enter.
    Или: Finder → Программы → Утилиты → Terminal
  3. 3
    Расположите окна Finder (папка installers/macos/) и Терминала рядом. В Терминале введите bash  (с пробелом в конце, без Enter), затем перетащите файл install.sh из Finder прямо в окно Терминала — путь вставится автоматически.
  4. 4
    Нажмите Enter и следуйте инструкциям установщика.
  5. 5
    Установщик создаст приложение «Bitrix24 + Фильтр чатов.app» в папке «Программы» и на рабочем столе.
⚠️
При первом запуске macOS может показать «Неизвестный разработчик». Откройте Системные настройки → Конфиденциальность и безопасность и нажмите «Открыть в любом случае».
Ручная установка

1. Скопируйте папку расширения в постоянное место, например:

~/Library/Application Support/PENA Agency

2. Откройте Терминал и запустите Bitrix24 с расширением (замените путь, если выбрали другое место):

"/Applications/Bitrix24.app/Contents/MacOS/Bitrix24" \
  --disable-extensions-except="$HOME/Library/Application Support/PENA Agency/Extension" \
  --load-extension="$HOME/Library/Application Support/PENA Agency/Extension"

3. Чтобы не вводить команду каждый раз, создайте ярлык на рабочем столе. Выполните в Терминале:

cat > ~/Desktop/Bitrix24-PENA.command << 'EOF'
#!/bin/bash
"/Applications/Bitrix24.app/Contents/MacOS/Bitrix24" \
  --disable-extensions-except="$HOME/Library/Application Support/PENA Agency/Extension" \
  --load-extension="$HOME/Library/Application Support/PENA Agency/Extension"
EOF
chmod +x ~/Desktop/Bitrix24-PENA.command

4. На рабочем столе появится файл Bitrix24-PENA.command. Дважды кликните по нему — Bitrix24 запустится с активным расширением. Можно перенести его в Dock или «Программы».

Первый запуск

  1. 1
    Запускайте Bitrix24 всегда через новый ярлык «Bitrix24 + Фильтр чатов» — только он передаёт параметры расширения.
  2. 2
    Перейдите в раздел Чаты.
  3. 3
    Панель фильтра чатов появится автоматически в верхнем левом углу списка чатов.
  1. 1
    Перейдите в раздел Чаты в Bitrix24.
  2. 2
    Панель фильтра чатов появится автоматически в верхнем левом углу списка чатов.
💡
Если панель не появилась — нажмите Ctrl + Alt + F (или ⌘ + ⌥ + F на Mac).
