# PENA Agency — Фильтр чатов Bitrix24

Расширение для Битрикс24, которое добавляет **панель фильтров** для списка чатов:
фильтрация по ключевым словам, тегам, категориям; группировка по датам в Открытых линиях.

---

## ✨ Возможности

- **Быстрые фильтры**: Непрочитанные / посмотреть позже / с вложениями
- **Поиск** по названию и последнему сообщению
- **Для Открытых линий**: WhatsApp / Telegram, статусы
- **Для чатов**: выбор типов (Диалоги, Группы, Телефон, Календарь, Общий и др.)
- **Для чатов задач**: скрытие завершённых, фильтр по тегам
- **Фильтры раздельны** для вкладок «Чаты» и «Чаты задач»
- **Теги-метки** — быстрая фильтрация по ключевым словам
- **Горячая клавиша**: `Ctrl + Alt + F` (macOS: `⌘ + ⌥ + F`)
- **Перетаскивание** по всему окну, позиция сохраняется

## 📦 Установка

### Windows (автоматически)
Запустите `installers\install_windows.bat` — установщик сделает всё сам.

### macOS (автоматически)
```bash
chmod +x installers/install_macos.sh && ./installers/install_macos.sh
```

### Браузер (вручную)
Откройте `installers/install_browser.html` и следуйте инструкции.

### Полная инструкция
Откройте `docs/guide.html` в браузере.

```

├── icons/                 — иконки (16, 48, 128px + logo)

│   ├── install_macos.sh
│   └── install_browser.html
└── docs/
    ├── guide.html         — полная инструкция
    └── screenshots/       — скриншоты
```

Расширение работает **локально**. Никакие данные чатов не отправляются — фильтрация
выполняется по DOM интерфейса Битрикс24.
