# Сортировщик чатов BX24 — для PENA Agency

Расширение для Битрикс24 (настольное приложение): добавляет **панель фильтров** над списком чатов.

---

## Возможности

- Быстрые фильтры: непрочитанные, «посмотреть позже», с вложениями
- Поиск по названию чата и последнему сообщению
- Фильтрация по типу (категории): диалоги, группы, телефон и др.
- Управление категориями — скрыть/показать встроенные, добавить собственные
- Теги-метки — фильтрация по ключевым словам
- **Пресеты фильтров** — сохранение и мгновенное переключение наборов фильтров
  - Режим отладки: позволяет редактировать фильтры активного пресета (защита от случайных изменений)
- Прозрачность панели, ресайз по боковым граням, перетаскивание
- Горячая клавиша: `Ctrl + Alt + F` (macOS: `⌘ + ⌥ + F`)
- Автообновление — ежедневная фоновая проверка новых версий (Windows)

## Установка

### Windows
Запустите `dist/PENA_Agency_Setup_v5.0.0.exe`.

Мастер установки создаст ярлык «Bitrix24 (PENA Agency)» в меню Пуск. На финальном экране — опциональные чекбоксы: создать ярлык на рабочем столе и запустить Bitrix24 сразу.

Альтернатива (без .exe): запустите `installers/windows/install.bat`.

### macOS — автономный бандл (рекомендуется)
1. Передайте пользователю файл `dist/BX24_Chat_Sorter_Mac.zip`
2. Пользователь разархивирует zip
3. Правый клик на `install.command` → **Открыть** (первый запуск)
4. Следовать инструкциям в терминале

Интернет не требуется — все файлы расширения уже внутри zip.

### macOS — онлайн-установщик
Дважды кликните `dist/Install_macOS.command` в Finder.
Скрипт скачает последнюю версию с GitHub и установит её.

### Браузер (Chrome, Edge, Яндекс, Brave)
Откройте `dist/Install_Browser.html` и следуйте инструкции.

## Передача пользователям

Папку `dist/` можно передавать отдельно — она содержит все файлы для установки:

| Файл | Назначение |
|------|-----------|
| `PENA_Agency_Setup_v5.0.0.exe` | Установщик Windows |
| `BX24_Chat_Sorter_Mac.zip` | Автономный бандл macOS (без интернета) |
| `Install_macOS.command` | Онлайн-установщик macOS (нужен интернет) |
| `Install_Browser.html` | Инструкция для браузерной версии |

Для нетехнических пользователей достаточно передать один файл:
- Windows → `PENA_Agency_Setup_v5.0.0.exe`
- macOS → `BX24_Chat_Sorter_Mac.zip`

## Структура проекта

```
extension/          — файлы Chrome-расширения (manifest.json, injected.js и др.)
installers/
  windows/          — install.bat, install.ps1, updater.ps1
  macos/            — install.command (онлайн), install_bundle.command (бандл), updater.sh
  build/            — setup.iss (Inno Setup), build.bat, build_mac_bundle.py
dist/               — готовые файлы для передачи пользователям
guide.html          — полная документация
update.json         — манифест версий для автообновления
```

## Выпуск обновления

1. Обновите код в `extension/`
2. Поднимите версию в `extension/manifest.json` и `update.json`
3. Пересоберите Windows `.exe`:
   ```
   installers/build/build.bat
   ```
4. Пересоберите Mac бандл:
   ```
   python installers/build/build_mac_bundle.py
   ```
5. Закоммитьте и запушьте:
   ```
   git add .
   git commit -m "feat: vX.Y.Z — описание"
   git push origin main
   ```
6. Создайте [GitHub Release](https://github.com/dmikhailovspace-commits/bx24-extension/releases) и прикрепите `.exe` и `.zip`

Пользователи Windows получат уведомление автоматически (Task Scheduler + OTA).
