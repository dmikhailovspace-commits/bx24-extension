# Сортировщик чатов BX24 — для PENA Agency

Расширение для Битрикс24, которое добавляет **панель фильтров** для списка чатов:
фильтрация по ключевым словам, категориям и тегам; сортировка; горячие клавиши.

---

## ✨ Возможности

- **Быстрые фильтры**: Непрочитанные / «посмотреть позже» / с вложениями
- **Поиск** по названию чата и последнему сообщению
- **Типы чатов**: Диалоги, Группы, Телефон, Открытые линии и др.
- **Теги-метки** — быстрая фильтрация по ключевым словам
- **Раздельные фильтры** для «Чаты» и «Чаты задач»
- **Прозрачность панели** — настраивается слайдером
- **Ресайз** по боковым граням панели
- **Горячая клавиша**: `Ctrl + Alt + F` (macOS: `⌘ + ⌥ + F`)
- **Автообновление** — ежедневная фоновая проверка новых версий

---

## 📦 Установка

### Windows
Запустите `dist/PENA_Agency_Setup_v5.0.0.exe` — мастер установки сделает всё сам.

### macOS
Дважды кликните `dist/Install_macOS.command` в Finder.
Скрипт скачает последнюю версию и установит её.

### Браузер (Chrome, Edge, Яндекс, Brave, Opera)
Откройте `dist/Install_Browser.html` в браузере и следуйте инструкции.

### Полная инструкция
Откройте `guide.html` в браузере.

## 📁 Структура проекта

```
extension/          — файлы Chrome-расширения (manifest.json, injected.js и др.)
installers/
  windows/          — установщик-скрипт (запасной вариант)
    install.bat     — запускает install.ps1
    install.ps1     — скрипт установки
    updater.ps1     — OTA-обновлятор (копируется на машину пользователя)
  macos/            — установщик для macOS
    install.command — двойной клик -> устанавливает расширение
    updater.sh      — OTA-обновлятор (копируется на машину пользователя)
  build/            — инструменты сборки установщика (для разработчиков)
    setup.iss       — исходник Inno Setup -> компилируется в .exe
    build.bat       — запускает ISCC.exe для сборки .exe
    build_pkg.sh    — сборка .pkg для macOS (запускать на Mac)
dist/               — готовые файлы для передачи пользователям
  PENA_Agency_Setup_v5.0.0.exe  — Windows-установщик
  Install_macOS.command         — macOS-установщик (скачивает с GitHub)
  Install_Browser.html          — руководство для установки в браузер
guide.html          — полная документация
update.json         — манифест версий для автообновления
```

---

## 🔄 Как выпустить обновление

1. Обновите код в `extension/`
2. Поднимите версию в `extension/manifest.json` и `update.json`
3. Пересоберите `.exe`: `installers/build/build.bat`
4. Закоммитьте и запушьте:
   ```
   git add .
   git commit -m "feat: vX.Y.Z — описание"
   git push origin main
   ```
Пользователи получат уведомление автоматически (Chrome + Windows Task Scheduler).
